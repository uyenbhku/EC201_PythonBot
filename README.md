
# Python Bot using Camunda 7, Gemini API

## By: [uyenbhku](https://github.com/uyenbhku)

### Prerequesites:
- Python 3.10
- Camunda Modeler


### Installations:
To run the repo on local or SaaS platforms: 

```
sh start.sh
```
Config the environment variables or `.env` file with all variables in `.envTemplate`.
